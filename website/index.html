Aura Recruit

Modern recruitment platform demo. Candidate management, job boards, and advanced filtering — built with Vue 3 and TypeScript.

Live Demo Source Code

Features

Candidate Management

Track and manage candidates through the hiring pipeline.

Job Board

Post and manage job listings with detailed descriptions.

Advanced Filtering

Filter candidates and jobs by skills, status, and more.

Vue 3 + TypeScript

Built with Vue 3 Composition API and full TypeScript support.

Build from Source

  1. Prerequisites: Node.js 20+, npm
  2. Clone: git clone https://github.com/cocodedk/aura-recruit-demo.git
  3. Install: npm install
  4. Run: npm run dev
